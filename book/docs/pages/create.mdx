# Create

Create a game.

::::steps

### Step one

Click the `Create` button

### Step two

Pop up window appears

### Step three

Choose a game name which will be displayed on the lobby list

### Step four

Select a time frame for the game

:::note
Note: if time is set to `0`, the game will be endless.
:::

### Step five

Click `Create` to initiate the lobby

### Step six

Click the `Join The Lobby` button
