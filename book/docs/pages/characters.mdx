# Characters

Every player has **7** characters. Each one can only be placed on specific structures. Each one represents a certain number of allegience points, which (along with how many points it scores) may increase on some structures.

## Lord & Lady ![lord-lady](lord-lady.png)

- can be placed on a `cities`, `roads` and `wonders`
- counts as `1` allegiance point
- no point scoring bonus

## Adventurer ![adventurer](adventurer.png)

- can be placed on a `cities`, `roads` and `wonders`
- counts as `2` allegience points on roads
- `2x` point scoring bonus per tile on a closed road

## Paladin ![paladin](paladin.png)

- can be placed on a `cities`, `roads` and `wonders`
- counts as `2` allegience points in cities
- `2x` point scoring bonus per tile in a closed city

## Pilgrim ![pilgrim](pilgrim.png)

- can be placed on a `cities`, `roads` and `wonders`
- counts as `1` allegience point
- `2x` total point scoring bonus when placed on a wonder

## Woodsman ![woodsman](woodsman.png)

- can onlt be placed on `forest`
- scores a `multiplier` for each adjoining closed road that borders it

## Herdsman ![herdsman](herdsman.png)

- can onlt be placed on `forest`
- scores a `multiplier` for each adjoining closed city that borders it

:::note
Note: `woodsman` & `herdsman` only score points when their forests are _**completely** enclosed.
:::
